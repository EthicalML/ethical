The AI-RFX Procurement Framework

A practical AI toolset to support key industry leaders

The AI-RFX Procurement Framework was put together to supports industry stakeholders when procuring AI systems by enabling them to assess projects through the Responsible Machine Learning Principles.

The AI-RFX consists of a set of templates to support a tender, project or competition. It implements our open source Machine Learning Maturity Model which is put together from best practices in the field of machine learning operations.

Raising the bar for AI safety, quality and performance

The AI-RFX framework has been put together by domain experts using best practices in industry for the design, devleopment and deployment of machine learning systems across industrial sectors.

This procurement framework goes beyond the ML algorithms themselves, and provides an assessment criteria to evaluate the maturity level of the technical and proces infrastructure around the machine learning algorithms.

Machine Learning Maturity Model

The Machine Learning Maturity Model (MLMM) is a set of criteria that ensures high-quality processes and technical infrastructure is in place where reasonable depending on the use-case. The MLMM was designed using the Responsible Machine Learning Principles, providing the ability for key industry stakeholders to set a bar of quality and safety when procuring machine learning solutions.

The 8 assessment criteria in the Machine Learning Maturity Model are:

  • Operational process design - Principle #1
  • Data and model assessment processes - Principle #2
  • Explainability by justification - Principle #3
  • Reproducible operations - Principle #4
  • Change management capability - Principle #5
  • Practical accuracy metrics - Principle #6
  • Trust by privacy - Principle #7
  • Security risk - Principle #8
